# StreamMate-By-Lumetix

#StreamMate-By-Lumetix# is a free stream companion for OBS that automates your titles, descriptions, tags, and overlays — so you can focus on streaming instead of setup.

## Features
- One-click metadata and overlay updates
- Preset system for recurring streams
- Scene-aware overlay automation
- Friendly, lightweight, and easy to use

## Installation
1. Download `streammate.lua` from this repository.
2. Open OBS → Tools → Scripts → + → Select `streammate.lua`.
3. Follow the in-dock instructions to set up your first preset.
4. Enjoy automated scene and overlay updates instantly.
